This package contains source files of an utility library of wrapped memory allocation and free functions that print out statistics summary to "stderr" periodically.

- Makefile consists of directives used by "make" tool to generate target/goals.
- myMemWrap.c is the C source file.


Note that a "libmemwrap.so" shared library will be generated by running the "make" utility with above-mentioned Makefile.


Example to test with the library by running this shell command:

	LD_PRELOAD=$PWD/libmemwrap.so find /usr -type f -exec cat '{}' \; > /dev/null
